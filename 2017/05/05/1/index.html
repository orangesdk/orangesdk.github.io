linux 开发环境安装(一)

发表于 2017-05-05

本文以ubuntu 14为例,介绍java基本开发环境多安装,步骤如下:

安装内容：

  1. 安装git
  2. 安装jenv
  3. 安装jdk
  4. 安装maven
  5. 安装idea

一、安装git

二、安装jenv

安装curl
安装jenv
更新jenv仓库

三、安装jdk

查看所有jdk版本
选择jdk 1.8.0_91版本进行安装

四、安装maven

查看所有maven版本
选择maven 3.3.3版本进行安装

五、安装idea

下载idea
进入压缩包所在目录,解压idea目录
建立软链接,本文idea目录(/home/winstar/ideaIU-2017.1.2/bin/idea),实际根据自己的idea目录
启动idea
